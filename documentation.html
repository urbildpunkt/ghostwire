Installation

Ghost themes are located in the ghost/content/themes/ directory.

To install your new theme:
  1. Copy the ghostwire folder into the ghost/content/themes/ directory.
  2. Restart Ghost.
  3. Login to your Ghost admin, and navigate to /ghost/settings/general/.
  4. Select the ghostwire theme in the 'Theme' options dropdown.
  5. Click 'Save'.

Templates
default.hbs

This is the default template which contains the HTML that appears on every page, including that of the header and footer.

The handlebars expression {{{body}}} denotes where the content from other templates which extend the default template goes.

Page templates start with {{!< default}} to specify that they extend the default template, and their content will be placed where {{{body}}} is defined in default.hbs.

index.hbs

This is the homepage template which extends default.hbs and defines how the posts listing will be displayed.

post.hbs

This template extends default.hbs and defines how each individual post page will be displayed.

partials/social-buttons.hbs - (Configuration & Usage)

This template defines the buttons that lead to your various social media accounts.

partials/comments.hbs - (Configuration & Usage)

This template sets up your Disqus commenting system.

Style Sheets
css/ghostwire.css

This style sheet contains all of the CSS compiled from the Sass source files.

css/custom.css

This style sheet is blank for adding in your own custom styles and overrides.

Config & Usage

Social Media Button Links

  1. Open the ghostwire/partials/social-buttons.hbs file.
  2. Replace the "#" of each button with your corresponsing social network profile link.
  3. Delete the full link of any social buttons that you don't want to use.
Full button link example: 
				<a href="#" target="_blank">
					<button class="btn btn-secondary twitter">
						<i class="fa fa-fw fa-twitter"></i>
					</button>
				</a>

Disqus Comments

  1. Open the ghostwire/partials/comments.hbs file.
  2. Replace 'ghostwire' from the var disqus_shortname = 'ghostwire'; variable with your own Disqus shortname.

See the Official Documentation for a more thorough installation guide.

Note: To remove the comments section completely, delete the {{> comments}} expression from ghostwire/post.hbs

Thumbnail Images

It is necessary to put the intended thumbnail image at the top of your post, which is a work-around until Ghost offers a better solution.

Note: Thumbnails will display up to five tags, with all but the first hidden until hovered over.

Header Background Images

Set the name of your image as featured-header and a jQuery hack will take care of the rest, another temporary work-around: ![featured-header]().

Note: When setting an image as the background of a post's header, it is best to put it at the very end of the post.

Recent Stories Widget

The recent stories list is generated from your blog's RSS feed using jQuery.

To change the number of posts displayed:
  1. Open the ghostwire/default.hbs file.
  2. Change '10' from the MaxCount: 10 variable at the bottom of the file.

Google Analytics

  1. Open the ghostwire/default.hbs file.
  2. See the bottom of the file for instructions on adding your site's ID.
